# Sync Environment

Manage environment variables with Vercel.

## Actions

### Pull from Vercel
- Get environment variables from Vercel
- Create/update .env.local
- Note any missing variables

### Compare Environments
- Check local vs Vercel variables
- Identify missing or extra variables
- Check for NEXT_PUBLIC_ prefix issues

### Quick Checks
- Verify production has all required vars
- Find undefined env references in code
- Check for exposed secrets
